VOICEVOX 青山龍星

VOICEVOXは「青山龍星」の重厚で低音な声で誰でも簡単に音声を作成できる、無料のテキスト読み上げソフトウェアです。

VOICEVOX

無料で使える中品質なテキスト読み上げソフトウェア

特徴

  • 商用・非商用問わず無料 (※1)
  • すぐに使えるソフトウェア (※2)
  • イントネーションの詳細な調整が可能

※1 詳しくは各キャラクターの利用規約をご参照ください

※2 Windows / Mac / Linux に対応
